### Get started
```bash
git clone https://github.com/siddheshkadane01/FinTrack-AI-Finance-Tracker.git
cd FinTrack-AI-Finance-Tracker
npm install
```

Add a `.env` file:
```env
DATABASE_URL=
DIRECT_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
GEMINI_API_KEY=
RESEND_API_KEY=
ARCJET_KEY=
```

Set up the database and run:
```bash
npx prisma db push
npm run dev
```

Open http://localhost:3000
